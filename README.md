🧠 Sobre o Projeto

O HVK Wallet foi desenvolvido com o objetivo de ajudar usuários a controlarem suas finanças de forma simples, visual e inteligente, permitindo inclusive o uso compartilhado entre múltiplas pessoas (casais, famílias, etc).

✨ Funcionalidades
💸 Controle Financeiro
Registro de entradas e saídas
Cálculo automático do saldo
Filtros por tipo de movimentação
Visualização compartilhada em tempo real

🔁 Automação Inteligente
Parcelamentos automáticos

📦 Como rodar o projeto localmente
# Clone o repositório
git clone https://github.com/seu-usuario/hvk-wallet.git

# Acesse a pasta
cd hvk-wallet

# Instale as dependências
npm install

# Rode o projeto
npm run dev

🔐 Configuração do Ambiente

Crie um arquivo .env com:

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key

📱 Instalação como App (PWA)

O HVK Wallet pode ser instalado no celular diretamente pelo navegador:

Acesse o link do app
Clique em "Adicionar à tela inicial"
Pronto! Ele funcionará como um app
